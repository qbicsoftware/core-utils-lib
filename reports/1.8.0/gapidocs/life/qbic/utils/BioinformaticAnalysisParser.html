class BioinformaticAnalysisParser

      Parser storing the fileTree of a nf-core pipeline output directory into JSON format


      Converts a fileTree generated by a nf-core pipeline into a JSON String similar to NanoporeParser

      Since:
      1.8.0
      Parameters:
      directory - path of nf-core directory whose fileTree should be converted into a JSON String
